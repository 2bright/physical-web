The Physical Web

The Physical Web is an experimental open source project to merge the web and physical worlds. The goal is to walk up to any smart object and use it with just a tap.

This application lets you find those devices. At this early stage in the project, you're not likely to see any unless you create them yourself. Go to this page to learn about how to create physical web beacons.

This is an early prototype so please, let us know if you have any problems. Go to the project page and report any issues you find.
