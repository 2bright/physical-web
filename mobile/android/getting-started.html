The Physical Web

Walk up and use anything

The Physical Web is an experimental open source project to merge the web and physical worlds.

This is an early prototype application lets you find and edit bluetooth beacons. You're not likely to see any unless you create them yourself. If have a Google test beacon, this walkthrough will get you started.

If you want are technical and want to create your own beacons, here is a list of devices you can program.

Please let us know if you have any problems. Go to the project page and report any issues you find.
